Convertisseur : retours à la ligne ↔ \n

Collez votre texte, puis utilisez Convertir pour transformer les retours à la ligne en \n, ou Inverser pour restaurer les retours à la ligne à partir d’un texte contenant des \n.

Remarques :
  • Convertir remplace les retours simples par \n et les doubles par \n \n.
  • Inverser effectue l’opération inverse, restituant les retours de ligne réels.
  • Le bouton “Ouvrir dans une nouvelle fenêtre” affiche le texte converti dans une page dédiée, pour faciliter le copier/coller.
